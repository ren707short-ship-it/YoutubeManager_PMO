## システム概要

YouTube Shorts 管理システムv2は、複数のクリエイター(外注)と協力してYouTube Shortsを量産運用するための統合管理ツールです。

- アカウント管理(YouTubeチャンネル管理)
- 動画タスク管理(6ステータス対応)
- アイデア管理
- アフィリエイトリンク管理
- ユーザー管理(外注プロフィール)
- フィードバックワークフロー

**v2で追加された新機能:**
- 🆕 **制作ポータル**(テンプレート・マニュアル統合)
- 🆕 **YouTube API連携**(自動データ取得)
- 🆕 **参考チャンネル分析**
- 🆕 **設定画面**(YouTube API設定)

## 制作ポータルの使い方

制作ポータルは、外注が制作に必要な情報(テンプレート・マニュアル)を一元管理する画面です。

### 3.1 テンプレートの管理

**テンプレートとは?**
- ジャンル別の編集テンプレート(CapCutプロジェクト)
- 台本のプリセット(日本語・英語)
- 制作時の注意事項

**テンプレート追加方法:**
1. 制作ポータル画面で「テンプレート追加」ボタンをクリック
2. 以下を入力:
   - **名前**: 「FANZA作品紹介_3カット」など
   - **カテゴリ**: fanza_intro / ai_love / celeb_gossip など
   - **説明**: このテンプレートの用途
   - **CapCutプロジェクトURL**: GoogleドライブなどのURL
   - **日本語台本テンプレート**: プレースホルダ形式の台本
   - **英語台本テンプレート**: 同上
   - **注意事項**: フォント、色、禁止事項など
3. 「作成」ボタンをクリック

**プレースホルダの例:**
```
【作品名】{{title}}
【出演者】{{actress}}
【ストーリー】{{story}}
```

### 3.2 マニュアルの管理

**マニュアルとは?**
- ジャンル別の運用ガイド
- コンプライアンスルール
- NG例・OK例

**マニュアル追加方法:**
1. 制作ポータル画面で「マニュアル追加」ボタンをクリック
2. 以下を入力:
   - **タイトル**: 「FANZA作品紹介_運用ガイド」など
   - **カテゴリ**: テンプレートと同じカテゴリを指定
   - **本文**: Markdown形式で記載(見出し、リスト、コードブロックが使えます)
3. 「作成」ボタンをクリック

**マニュアル例:**
```markdown
# FANZA作品紹介_運用ガイド

## コンプライアンス

### 必須ルール
- **作品名は必ず伏せ字**: 「○○○の秘密」のように表記
- **出演者名も伏せ字**: 「○○○ちゃん」
- **直接的な表現禁止**: 遠回しな表現を使用

### NG例
❌ 「〇〇(実名)の新作！」

### OK例
✅ 「人気女優○○の最新作」
```

### 3.3 カテゴリ管理

カテゴリは自由に追加できます。以下のような命名規則を推奨します:
- `fanza_intro` - FANZA作品紹介
- `ai_love` - AI恋愛
- `celeb_gossip` - 芸能人ゴシップ
- `travel_spots` - 旅行スポット紹介
- `anime_review` - アニメレビュー

## YouTube API連携

### 4.1 自アカウント動画の同期

**機能:**
- YouTubeチャンネルから最新50件の動画を自動取得
- 再生数・いいね数を更新
- 新しい動画を「投稿済み」として自動登録

**使い方:**
1. アカウント管理画面で、対象アカウントの「YouTube同期」ボタンをクリック
2. 同期が完了すると、動画一覧に反映されます
3. 設定画面の「同期履歴」で結果を確認できます

**注意事項:**
- アカウントのURLフィールドにYouTubeチャンネルURLが登録されている必要があります
- APIキーが設定され、有効化されている必要があります

### 4.2 参考チャンネルの分析

**機能:**
- 成功している他チャンネルの動画データを取得
- 再生数順で人気動画を分析
- タイトル・説明・再生回数・長さを記録

**使い方:**
1. 参考チャンネル管理画面で「参考チャンネル追加」
2. チャンネル名とYouTube Channel IDを入力
3. 「同期」ボタンで動画データを取得
4. 人気動画を一覧表示し、成功パターンを分析

**YouTube Channel IDの取得方法:**
- チャンネルページのURLから: `youtube.com/channel/UCxxxxxx` の `UCxxxxxx` 部分
- または: `youtube.com/@channelname` → ページのソースから取得

## 運用フロー

### 5.1 オーナーの日次業務

**朝(計画):**
1. ダッシュボードで全体状況を確認
   - フィードバック待ち動画
   - 期限が近い動画
   - クリエイター別統計
2. アイデア管理でネタを選定
3. 選んだネタから動画タスクを作成
   - テンプレートを選択
   - 担当クリエイターをアサイン
   - 期限を設定

**夕方(確認・フィードバック):**
1. ステータスが「確認待ち」の動画をチェック
2. コメント機能でフィードバック
3. 問題なければステータスを「予約済み」または「投稿済み」に変更
4. フィードバックが必要な場合:
   - フィードバック必須フラグをON
   - フィードバック期限を設定
   - 具体的な修正内容をコメントで指示

**週次:**
1. ユーザー管理画面でクリエイター別パフォーマンスを確認
2. 参考チャンネルの人気動画を分析
3. テンプレート・マニュアルを更新

### 5.2 動画タスク作成のポイント

**テンプレートから作成:**
1. 動画新規作成時に「テンプレートから作成」を選択
2. ジャンルに合ったテンプレートを選ぶ
3. 台本テンプレートが自動的に入力される
4. プレースホルダ({{title}}など)を実際の内容に置き換える

**必須項目:**
- タイトル
- アカウント
- 担当クリエイター
- テンプレート
- 期限

**推奨項目:**
- 台本(日本語・英語)
- アセットリンク(素材のGoogleドライブURLなど)
- アフィリエイトリンク(該当する場合)

## 外注管理のベストプラクティス

### 6.1 新しい外注を追加する時

1. **ユーザー作成:**
   - 名前、メールアドレス、パスワード
   - ロール: Creator
   - 契約プラットフォーム: クラウドワークス / ランサーズ / ココナラ / 直接
   - 契約日: YYYY-MM-DD形式
   - 契約書類URL: GoogleドライブなどのURL
   - ステータス: アクティブ

2. **提供マテリアル登録:**
   - ユーザー詳細画面で「マテリアル追加」
   - CapCutテンプレート、素材集、台本例などを登録
   - URLで共有(Googleドライブ推奨)

3. **初回オリエンテーション:**
   - システムのログイン方法を共有
   - 制作ポータルの使い方を説明
   - 最初の動画タスクをアサイン

### 6.2 外注への指示のコツ

**明確な期限設定:**
- 「今週中」ではなく「2025-01-15 18:00」のように具体的に

**テンプレート・マニュアルの徹底:**
- 必ず制作ポータルのマニュアルを読んでもらう
- 初回はオーナーが一緒に画面を見ながら説明

**フィードバックは具体的に:**
❌ 「もっとよくして」
✅ 「2:30の場面で、テロップの色を白→黄色に変更してください」

**定期的なコミュニケーション:**
- コメント機能を活用
- 週1回は全体の状況を共有

### 6.3 問題発生時の対応

**動画が削除された場合:**
1. 該当動画のステータスを記録
2. どのルールに違反したか分析
3. マニュアルを更新
4. 全クリエイターに通知

**クリエイターのパフォーマンスが低い場合:**
1. ユーザー詳細画面で統計を確認
2. 1on1でフィードバック
3. 必要に応じてステータスを「一時停止」に変更

## よくある質問

**Q: テンプレートとマニュアルの違いは?**
A: テンプレートは編集ファイルや台本のプリセット。マニュアルは運用ルールやNG例の説明書です。

**Q: YouTube API連携は必須ですか?**
A: いいえ、手動で動画データを入力することもできます。APIを使うと自動化されるだけです。

**Q: クリエイターは何が見えますか?**
A: 自分に割り当てられたアカウント・動画、制作ポータル(テンプレート・マニュアル)が見えます。アイデア、アフィリエイト、ユーザー管理、設定画面はオーナー専用です。

**Q: バックアップは?**
A: システムは定期的にバックアップされますが、重要なデータ(台本、アイデア)は別途エクスポートすることを推奨します。

## トラブルシューティング

**ログインできない:**
- メールアドレス・パスワードを確認
- ブラウザのCookieを有効化

**YouTube API同期でエラー:**
- APIキーが正しいか確認
- APIキーのクォータ制限を確認(無料枠は1日10,000リクエスト)
- YouTube Data API v3が有効化されているか確認

**テンプレートが表示されない:**
- カテゴリが正しく設定されているか確認
- ブラウザを再読み込み
